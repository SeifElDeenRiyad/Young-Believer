# Young Believer

Young Believer is an Islamic learning app designed primarily for children to help build faith, understanding, and identity through simple, structured, and accessible content.

## Purpose

Young Believer aims to make learning Islam simple, structured, and accessible while helping raise a stronger and more aware generation of Muslims.

## Target Audience

- Primarily designed for children
- Accessible and usable for all ages

## Overview

The app provides a focused learning experience through organized Islamic content that users can explore section by section. It is built to be simple to navigate, easy to understand, and suitable for regular use without unnecessary complexity.

## Features

### Current Features

- Holy Quran
- Duas
- Azkar
- Prophets' Stories
- Asma Allah Al-Husna
- Pillars of Islam

### Current State

- All listed sections are functional
- A usable APK is available for testing and installation
- The app offers an offline-first experience
- No sign-in is currently required

## Vision

Young Believer is being developed with the goal of creating a more personalized and engaging learning journey for each child, based on:

- Interaction with content
- Learning pace
- Simplicity and clarity

## Usage

1. Open the app
2. Choose a section from the home screen
3. Read, explore, and navigate the available content

## Tech Stack

- Java
- XML
- Android Studio
- Gradle
- Android SDK
- AndroidX libraries
- Material Components

**Minimum Android Version:** Android 7.0 (API 24)

## How to Run

### Run on Device or Emulator

- Open the project in Android Studio
- Allow Gradle to sync
- Click **Run** and choose an emulator or connected Android device

### Build APK

- In Android Studio, go to `Build > Build APK(s)`
- The generated debug APK will be located at:

`app/build/outputs/apk/debug/app-debug.apk`
